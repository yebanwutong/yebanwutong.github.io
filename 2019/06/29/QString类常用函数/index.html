QString类常用函数

发表于 2019-06-29 更新于 2019-10-10 分类于 C/C++

QString类提供了一个Unicode文本和经典的C以零结尾的字符数组的抽象。

  1. QString::QString()
    构造一个零字符串。这是一个不被分配任何东西的字符串，也就是说长度和数据指针都为0。

  2. bool QString::isNull() const
    如果字符串为零，返回真。零字符串总是空的。

  1. bool QString::isEmpty() const
    如果字符串为空，也就是如果length() == 0，返回真。因此，零字符串也是空字符串。

  1. uint QString::length() const
    返回字符串的长度。零字符串和空字符串的长度都为0

  2. QString & QString::append(const QString & str)
    把str添加到字符串中并且返回结果的引用

  1. QString QString::arg(const QString & a, int fieldwidth = 0) const
    返回使用a来替换最低层出现的%i（i为’1’或’2’或……’9’）的字符串。
    fieldwidth值指定了填充到a中的最小空间。正值将产生右对齐文本，负值将产生左对齐文本。
  1. QChar QString::at(uint i) const
    返回在索引i处的字符，或者如果i超过字符串的长度返回0。

  1. int static QString::compare(const QString & s1, const QString & s2)
    对s1和s2进行词典比较，如果s1小于、等于或者大于s2，就返回小于、等于或者大于0的整数。

  2. QString QString::lower() const
    返回由这个字符串转换的小写字符串。

  3. QString static QString::number (long n, int base = 10)
    一个把数字n转换为字符串的方便函数，n被基于base表示，默认为10，并且必须在2到36之间。

  4. QString QString::trimmed() const
    移除字符串两端的空白字符。

  5. QString QString::simplified() const
    移除字符串两端的空白字符。字符串中间的空格（包括单个空格、多个空格、\t、\n）都统一转化成一个空格
