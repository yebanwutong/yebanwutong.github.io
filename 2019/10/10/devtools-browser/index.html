Chrome Devtools Browser

发表于 2019-10-10 分类于 chrome devtools protocol

浏览器域定义用于浏览器管理的方法和事件

Methods

Browser.setPermission

设置给定来源的权限设置[实验性质]

参数

  • origin string 许可适用的来源。
  • permission PermissionDescriptor 覆盖权限的描述符。
  • setting PermissionSetting 权限的设置。
  • browserContextId[optional] Target.TargetID 要覆盖的上下文。 如果省略，则使用默认浏览器上下文

Browser.grantPermissions

授予给定来源特定的权限，拒绝所有其他权限[实验性质]

参数

  • origin string 许可适用的来源。
  • permissions array PermissionDescriptor 覆盖权限的描述符。
  • browserContextId[optional] Target.BrowserContextID BrowserContext覆盖权限。 如果省略，则使用默认浏览器上下文

Browser.resetPermissions

重置所有来源的所有权限管理[实验性质]

参数

  • browserContextId[optional] Target.BrowserContextID BrowserContext重置权限。 如果省略，则使用默认浏览器上下文。

Browser.close

正常关闭浏览器

Browser.crash

在主线程上使浏览器崩溃[实验性质]

Browser.crashGpuProcess

导致GPU进程崩溃[实验性质]

Browser.getVersion

返回版本信息

返回

  • protocolVersion string 协议版本
  • product string 产品名称
  • revision string 产品版本
  • userAgent string User-Agent
  • jsVersion string v8引擎版本

Browser.getBrowserCommandLine

仅当且仅当返回浏览器进程的命令行开关时，

  • –enable-automation
    [实验性质]

返回

  • arguments array [string] 命令行参数

Browser.getHistograms

获取Chrome直方图[实验性质]

参数

  • query[optional] string 请求的子字符串名称。 仅提取名称中带有查询作为子字符串的直方图。 空查询或缺席查询将返回所有直方图。
  • delta[optional] boolean 如果为true，则检索自上次调用以来的增量。

返回

  • histograms array Histogram 直方图

Browser.getHistogram

按名称获取Chrome直方图[实验性质]

参数

  • name string 要求的直方图名称
  • delta[optional] boolean 如果为true，则检索自上次调用以来的增量。

返回

  • histogram Histogram 直方图

Browser.getWindowBounds

获取浏览器窗口的位置和大小[实验性质]

参数

  • windowId WindowID 浏览器窗口ID

返回

  • bounds Bounds 窗口的边界信息。 当窗口状态为“最小化”时，将返回还原的窗口位置和大小。

Browser.getWindowForTarget

获取包含devtools目标的浏览器窗口[实验性质]

参数

  • targetId[optional] Target.TargetID Devtools代理主机标识。 如果作为会话的一部分进行调用，则使用关联的targetId。

返回

  • windowId WindowID 浏览器窗口ID
  • bounds Bounds 窗口的边界信息。 当窗口状态为“最小化”时，将返回还原的窗口位置和大小

Browser.setWindowBounds

设置浏览器窗口的位置和/或大小[实验性质]

参数

  • windowId WindowID 浏览器窗口ID
  • bounds Bounds 新窗口边界。 “最小化”，“最大化”和“全屏”状态不能与“左”，“上”，“宽度”或“高度”组合使用。 保留未指定的字段不变。

Browser.setDockTile

设置特定平台的停靠图块详细信息[实验性质]

参数

  • badgeLabel[optional] string
  • image[optional] string Png编码的图像

Types

WindowID

[实验性质]

Type: integer

WindowState

浏览器窗口的状态[实验性质]

Type: string

允许值: normal, minimized, maximized, fullscreen

Bounds

浏览器窗口边界信息[实验性质]

Type: Object

数据结构

  • left[optional] integer 从屏幕左边缘到窗口的偏移量（以像素为单位）
  • top[optional] integer 从从屏幕顶部到窗口的偏移量（以像素为单位）
  • width[optional] integer 窗口宽度（以像素为单位）
  • height[optional] integer 窗口高度（以像素为单位）
  • windowState[optional] WindowState 窗口状态。 默认为normal

PermissionType

[实验性质]

Type: string

允许值: accessibilityEvents, audioCapture, backgroundSync, backgroundFetch, clipboardRead, clipboardWrite, durableStorage, flash, geolocation, midi, midiSysex, notifications, paymentHandler, periodicBackgroundSync, protectedMediaIdentifier, sensors, videoCapture, idleDetection, wakeLockScreen, wakeLockSystem

PermissionSetting

[实验性质]

Type: string

允许值: granted, denied, prompt

PermissionDescriptor

Permissions API中定义的PermissionDescriptor的定义：https://w3c.github.io/permissions/#dictdef-permissiondescriptor[实验性质]

Type: Object

数据结构

  • name string 许可名称。 有关有效的权限名称，请参见https://cs.chromium.org/chromium/src/third_party/blink/renderer/modules/permissions/permission_descriptor.idl。
  • sysex[optional] boolean 对于“ midi”权限，还可以指定sysex控件。
  • userVisibleOnly[optional] boolean 对于“push”权限，可以指定userVisibleOnly。 请注意，userVisibleOnly = true是当前唯一支持的类型。
  • type[optional] string 对于“wake-lock”权限，必须将类型指定为“screen”或“system”。

Bucket

铬直方图桶[实验性质]

Type: Object

数据结构

  • low integer 最小值（包含当前值）。
  • high integer 最大值（包含当前值）。
  • count integer 样本数

Histogram

铬直方图[实验性质]

Type: Object

数据结构

  • name string 名称
  • sum integer 样本值的总和
  • count integer 样本数
  • buckets array Bucket
