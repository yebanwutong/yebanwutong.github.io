序列

发表于 2019-03-13 更新于 2019-10-10 分类于 Python

python包含六种内建的序列（列表、元组、字符串、Unicode字符串、buffer对象和xrange对象）

索引

序列中所有元素的下标都是从0开始递增

分片

与使用索引类似，可以使用分片操作来访问一定范围的元素
